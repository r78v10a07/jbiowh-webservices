GenePTT access



Get the number of GenePTT available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/count



Get one GenePTT by ProteinGi:

    http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/#{dataSets.genePTTProteinGi}



Get from/to GenePTT by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/1/3



Get GenePTTs from a PTT file:

    http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/chromosome/NC_009504



Get GenePTTs from a PTT file using from/to range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/chromosome/NC_009504/700077/744610



Search GenePTTs by string using % (Search over the fields: ProteinGi, PTT File name, Gene symbol, locustag, in this order. Use %25 for the wildcard):



    ProteinGi:

       http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/search/#{dataSets.genePTTProteinGi}

    PTT File name:

       http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/search/NC_01739%25

    Gene symbol:

       http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/search/4hbD

    Locustag:

       http://#{dataSets.server}/jbiowh-webservices/webservices/geneptt/search/AM1_H0113
