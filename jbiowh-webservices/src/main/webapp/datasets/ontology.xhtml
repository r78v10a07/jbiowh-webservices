Ontology access



Get the number of Ontologies available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/ontology/count



Get one Ontology by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/ontology/#{dataSets.ontologyWID}



Get from/to Ontologies by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/ontology/1/3



Get Ontologies by GO ID (the % wilcard (%25) is available):

    http://#{dataSets.server}/jbiowh-webservices/webservices/ontology/goid/GO:00000%25



Search the Ontologies by string (Search over the fields: GO ID, name, definition, comment, synonym, alternative id, in this order. Use %25 for the wildcard):

    http://#{dataSets.server}/jbiowh-webservices/webservices/ontology/search/%25human%25
