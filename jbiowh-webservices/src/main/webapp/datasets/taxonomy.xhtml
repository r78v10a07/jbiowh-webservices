Taxonomy access



Get the number of Taxonomies available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/count



Get one Taxonomy by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/#{dataSets.taxonomyWID}



Get from/to Taxonomies by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/1/3



Get the Taxonomy by TaxId:

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/taxid/9606



Search Taxonomies by string using % (search all "Bos*"):

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/search/bos%25



External data links

GeneInfo

Get Genes by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/#{dataSets.taxonomyWID}/geneinfo



Get Genes by Tax Id (Example Tax id: 670, Vibrio parahaemolyticus):

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/taxid/9906/geneinfo



Protein

Get Proteins by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/#{dataSets.taxonomyWID}/protein



Get Genes by Tax Id (Example Tax id: 670, Vibrio parahaemolyticus):

    http://#{dataSets.server}/jbiowh-webservices/webservices/taxonomy/taxid/9906/protein
