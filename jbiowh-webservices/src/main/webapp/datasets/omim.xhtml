OMIM access



Get the number of OMIMs available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/omim/count



Get one OMIM by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/omim/#{dataSets.OMIMWID}



Get from/to OMIMs by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/omim/1/3



Get OMIM by OMIM Id:

    http://#{dataSets.server}/jbiowh-webservices/webservices/omim/id/100300



Search OMIMs by string using % (Search over the fields: OMIM ID and Title in this order. Use %25 for the wildcard):



    OMIM ID:

       http://#{dataSets.server}/jbiowh-webservices/webservices/omim/search/100500

    Title (*AUTOSOMAL*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/omim/search/%25AUTOSOMAL%25
