GenBank access



Get the number of GenBank available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/genbank/count



Get one GenBank entry by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/genbank/#{dataSets.geneBankWID}



Get from/to GenBanks by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/genbank/1/3



Get GenBank By Gi:

    http://#{dataSets.server}/jbiowh-webservices/webservices/genbank/gi/#{dataSets.geneBankGi}



Get GenBank by LocusName:

    http://#{dataSets.server}/jbiowh-webservices/webservices/genbank/locusname/CP002312



Search GeneBank entries by string using % (Search over the fields: ProteinGi, definition in this order. Use %25 for the wildcard):



    ProteinGi:

       http://#{dataSets.server}/jbiowh-webservices/webservices/geneinfo/search/#{dataSets.geneBankProteinGi}

    Definition:

       http://#{dataSets.server}/jbiowh-webservices/webservices/geneinfo/search/Borrelia%20burgdorferi%20JD1
