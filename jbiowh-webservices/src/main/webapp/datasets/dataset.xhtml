DataSet access



Get the number of DataSet available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/count



Get all DataSet available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/



Get one DataSet by Id:

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/1



Get from/to DataSets by Id:

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/1/3



Get DataSets by Name (the dataset with name: Taxonomy):

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/name/Taxonomy



Get DataSets by Name using % (all datasets with o):

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/name/%25o%25



Get DataSets by Name using % and negative selection (all datasets without o):

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/noname/%25o%25



Get DataSets by Status (all datasets with status: inserted):

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/status/inserted



Get DataSets by Status using % (all datasets with e):

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/status/%25e%25



Get DataSets by status using a negative selection (all datasets with status is not: inserted):

    http://#{dataSets.server}/jbiowh-webservices/webservices/dataset/nostatus/inserted
