PirSF access



Get the number of Pirsf available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/pirsf/count



Get Pirsf by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/pirsf/#{dataSets.PIRSFWID}



Get from/to Pirsf by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/pirsf/1/3



Get from/to Pirsf by PirSFNumber:

    http://#{dataSets.server}/jbiowh-webservices/webservices/pirsf/pirsfnumber/PIRSF038885



Search Pirsf groups by string using % (Search over the fields: PirSFNumber, name and accession number in this order. Use %25 for the wildcard):



    PirSFNumber (PIRSF03888*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/pirsf/search/PIRSF03888%25

    Name (cytochrome*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/pirsf/search/cytochrome%25

    Accession Number (A0A174):

       http://#{dataSets.server}/jbiowh-webservices/webservices/pirsf/search/A0A174
