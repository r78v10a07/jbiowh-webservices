Protein access



Get the number of Proteins available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/protein/count



Get Protein by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/protein/#{dataSets.proteinWID}



Get from/to Protein by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/protein/1/3



Get Proteins by Name (040R_*):

    http://#{dataSets.server}/jbiowh-webservices/webservices/protein/name/040R_%25



Get Proteins by Accession (Q197A*):

    http://#{dataSets.server}/jbiowh-webservices/webservices/protein/accession/Q197A%25



Search Proteins by string using % (Search over the fields: name, EC, accession number and complete name in this order. Use %25 for the wildcard):



    Name (040R_*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/protein/search/040R_%25

    EC (3.*.*.89):

       http://#{dataSets.server}/jbiowh-webservices/webservices/protein/search/3.%25.%25.89

    Accession Number (Q197A*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/protein/search/Q197A%25

    Complete Name (Uncharacterized*protein*Gp24*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/protein/search/Uncharacterized%25protein%25Gp24%25
