DrugBank access



Get the number of DrugBanks available:

    http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/count



Get one DrugBank by Id (this Id is the internal JBioWH WID that change in all DB):

    http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/#{dataSets.drugBankWID}



Get from/to DrugBanks by range:

    http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/1/3



Get DrugBank by DrugBank Id:

    http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/id/DB00009



Search DrugBanks by string using % (Search over the fields: DrugBank ID, CAS number, name and indication in this order. Use %25 for the wildcard):



    DrugBank ID (DB0000*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/search/DB0000%25

    CAS number (*-60-*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/search/%25-60-%25

    Indication (*asthma*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/search/%25asthma%25

    Name (Prop*):

       http://#{dataSets.server}/jbiowh-webservices/webservices/drugbank/search/prop%25
